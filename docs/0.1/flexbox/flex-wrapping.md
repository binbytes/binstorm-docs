# Flex Wrapping
### Utilities for controlling how flex items wrap.

|Class|Properties|
|-|-|
|flex-no-wrap|flexWrap: 'nowrap'|
|flex-wrap|flexWrap: 'wrap'|
|flex-wrap-reverse|flexWrap: 'wrap-reverse'|


### Usage
Control the flex wrap of view using the `flex-{wrap}` utilities.
