# Flex - Grow & Shrink
### Utilities for controlling how flex items grow and shrink.

|Class|Properties|
|-|-|
|flex-grow|flexGrow: 1|
|flex-no-grow|flexGrow: 0|
|flex-shrink|flexShrink: 1|
|flex-no-shrink|flexShrink: 0|
